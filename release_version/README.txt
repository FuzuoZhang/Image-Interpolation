程序运行环境：
操作系统：Windows10
运行环境为同时配置有 Qt 5.9.2 以及opencv-2.4.13.2的vs 2017

使用的库：
opencv-2.4.13.2实现图片的读取和存储
Qt写程序界面

操作方法:
1）先进入主界面->点击左上方的按钮选择一张图片->选择对图像变换的方式
   注：请不要选择路径中带有中文字符的图片。
2）进入具体的图像变换界面
若为旋转变换：
  通过微调框和滑条改变R的范围（1-256）和θ（0，2π） 
  选择具体的插值方式
  单击“确定”按钮

若为水波纹变换：
  通过微调框和滑条改变R的范围（0-400），φ（0，2π），ρ（0，100）
  选择具体的插值方式
  单击“确定”按钮

若为B样条变形：图片上均匀分布有9个控制点
  在图片上改变鼠标位置，当鼠标处在某一控制点附近时鼠标会变为十字形
  当鼠标为十字形时可以按下鼠标左键，并拖动鼠标到另一位置
  可以拖动多个点，同一个点可拖动多次
  选择插值方法，点击“确定”按钮

3）经过2）中的步骤即可得到一张变换后图片，点击左下角的“点我保存”按钮，可将变换后的图片保存下来。
